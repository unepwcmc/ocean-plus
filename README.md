# ocean-plus

> A Vue.js project

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

## Deploying

Before deploying, run `yarn build` and commit and push the results to the branch you wish to deploy.
